🏥 Hospital Appointment Booking System
A robust Full-Stack Web Application designed to streamline the process of scheduling medical appointments. This project bridges the gap between patients and healthcare providers by providing a clean, user-friendly interface for booking slots and a reliable backend to manage data.

🚀 Overview
This system allows users to book medical appointments online by filling out a digital form. The data is processed through a secure backend API and stored in a cloud database, replacing traditional manual record-keeping with an automated, digital solution.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Backend: Node.js, Express.js

Database: MongoDB (via Mongoose ODM)

API Testing: Postman / Logic testing

Tools: Cursor AI, Git, GitHub

✨ Key Features
Interactive Booking Form: Users can enter their name, contact details, and preferred appointment time.

Department Selection: Dynamic choice of medical departments (e.g., Cardiology, Orthopedics, General Medicine).

Real-time Data Storage: Appointments are instantly saved to a MongoDB cluster.

Responsive Design: Fully functional on desktops, tablets, and mobile phones.

RESTful API: A structured backend that handles POST requests to create new appointments and GET requests to retrieve them.
